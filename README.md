# 🏠 Melbourne House Price Predictor

A machine learning web app that predicts Melbourne property prices
using a Gradient Boosting model trained on 13,580 sales records.

## Results
- Best Model: Gradient Boosting
- R² Score: 0.837
- MAE: ~$149,000

## Steps
1. Data Preprocessing & EDA
2. Feature Engineering
3. Model Training & Evaluation
4. Streamlit Deployment

## How to Run
pip install -r requirements.txt
streamlit run app.py

## Dataset
Melbourne Housing Snapshot — Kaggle (dansbecker)
